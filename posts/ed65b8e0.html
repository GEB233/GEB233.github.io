hexo-插件升级

  • 如何更新Hexo及插件的版本？如何更新主题版本？ | 张洪Heo (zhheo.com)

1 自动升级

升级大部分插件，基本模块不升级

博客根目录内：

npm update

2 手动升级

博客根目录内：

  1. 检查更新

    npm outdated

  2. 更新版本文件

    改根目录的package.json文件，将对应插件名称所对应的版本号更改为要更新的版本号。

    "dependencies": {
   "hexo": "^6.2.0",
   "hexo-abbrlink": "^2.2.1",
   "hexo-baidu-url-submit": "0.0.6",
   "hexo-deployer-git": "^3.0.0",
   "hexo-filter-github-emojis": "^3.0.5",
   "hexo-generator-archive": "^1.0.0",
   "hexo-generator-baidu-sitemap": "^0.1.9",
   "hexo-generator-category": "^1.0.0",
   "hexo-generator-feed": "^3.0.0",
   "hexo-generator-index": "^2.0.0",
   "hexo-generator-json-content": "^4.2.3",
   "hexo-generator-search": "^2.4.3",
   "hexo-generator-sitemap": "^3.0.1",
   "hexo-generator-tag": "^1.0.0",
   "hexo-helper-qrcode": "^1.0.2",
   "hexo-lazyload-image": "^1.0.10",
   "hexo-permalink-pinyin": "^1.1.0",

  3. 开始更新

    npm install --save

3 升级报错

一般出现类似错误是由于升级hexo后，其余依赖未升级完全，可以考虑删除依赖，重新 install。

npm cache clean --force
rm -rf node_modules
rm package-lock.json
npm install
