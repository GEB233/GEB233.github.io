1 分析基础

1.1 类似泰勒展开观点

​ 泰勒展开为多项式展开，所利用的基本函数为多项式函数。由其形式所带来的启发是，做到函数的数值近似，应满足的条件为：

  • 数值相等

  • 一阶导数相等

  • 二阶及高阶导数相等

对F(x),使用指定函数G(x)的伸缩平移来拟合。aG(x+b)+c \[ \left\{ \begin{matrix} {F\left( x_{0} \right) = aG\left({x_{0} + b} \right) + C} \\ {F^{'}\left( x_{0} \right) = aG^{'}\left({x_{0} + b} \right.}) \\ {F{''}\left( x_{0} \right) = aG{''}\left( x_{0} + b \right)} \\ \end{matrix} \right. \]

\[ \mathbf{由}\frac{F^{''}\left( x_{0} \right)}{F^{'}\left( x_{0} \right)} = \frac{G^{''}\left( t \right)}{G^{'}\left( t \right)}~~,\mathbf{解得}t\hspace{25em}\\ \mathbf{则}\left\{ \begin{matrix} {~b = t - x_{0}~~} \\ {a = \frac{F^{'}\left( x_{0} \right)}{G^{'}\left( t \right)}} \\ {c = F\left(x_{0} \right) - aG\left(t \right.} )\\ \end{matrix} \right. \]

因此 \[ F = F\left( x_{0} \right) + \frac{F^{'}\left( x_{0} \right)}{G^{'}\left( t \right)}\left( G\left( {x - x_{0} + t} \right) - G\left( t \right) \right) \]

\[ 当G为反比例函数\frac{1}{x+1}时\hspace{25em}\\ 记~~~T=\frac{F^{'}(x_0)}{F^{''}(x_0)}~~~P=x-x_0\\ 则有F = F_{x_{0}} + \frac{2TP}{2T - P}F_{x_{0}}^{'} \]

1.3 旋转

1 轴对称点：在定区间内，视函数曲率最大值点为函数轴对称点。 \[ k = \frac{\left| f_{2} \right|}{\left\lbrack {1 + f_{1}^{2}} \right\rbrack^{\frac{3}{2}}} \Rightarrow f_{3} = \frac{3f_{1}f_{2}^{2}}{1 + f_{1}^{2}}\\[5pt] \] 其中f1,f2,f3为函数的一二三阶导数,方程解即为函数轴对称点。

2 变换 设对称点处斜率k，则对称点到原点的等距旋转变换为： \[ \left\{ \begin{matrix} {x = \frac{ky^{'} - x'}{\sqrt{1 + k^{2}}}} \\[5pt] {y = \frac{kx^{'} + y'}{\sqrt{1 + k^{2}}}} \\ \end{matrix} \right. \] 上式作用，将（x,y）待入原函数，得到对称轴为y轴的新方程。

一般的，旋转θ: \[ \begin{pmatrix} {x'} \\ {y'} \\ \end{pmatrix} = \begin{pmatrix} \begin{matrix} {cos\theta} & {- sin\theta} \\ \end{matrix} \\ \begin{matrix} {sin\theta} & {cos\theta} \\ \end{matrix} \\ \end{pmatrix}\begin{pmatrix} x \\ y \\ \end{pmatrix} \]

2.3 指数函数近似

2.3.1 根号指数

\[ \sqrt[n]{a^{n} + b} \geq a + \frac{2ab}{2na^{n} + \left( {n - 1} \right)b}~~~~~b > 0 \]

​ 定义误差值 \[ \Delta = 真值 - 估值\\ b_{max} = \left( {a + 1} \right)^{n} - a^{n}\\ \Delta_{max} = \Delta_{b_{max}} = 1 - \frac{2a}{\frac{2na^{n}}{\left( {a + 1} \right)^{n} - a^{n}} + \left( n - 1 \right)} \] ​ 给出误差限 \[ \Delta \leq \frac{b}{b_{max}}\Delta_{max}\\ 令t=\frac{2a}{n-1}\\ \Delta_{max}\leq\frac{1}{t^2+t+1}\leq\frac{1}{(t+\frac{1}{2})^2}\leq\frac{1}{t(t+1)}\leq(\frac{n-1}{2a})^2_{(t\geq4时,差值小)} \] ​ 常用近似式 \[ \sqrt{a^{2} + b} = a + \frac{2ab}{4a^{2} + b}~~~,~~~\Delta < \frac{b}{2a\left( {2a + 1} \right)^{2}}\\ \sqrt[3]{a^{3} + b} = a + \frac{ab}{3a^{3} + b}~~~~,~~~\Delta < \frac{b}{4a^{2}\left( {a + 1} \right)^{2}} \]
